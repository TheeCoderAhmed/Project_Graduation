## 1. Project Overview

This application allows patients in Turkey to search for doctors and pharmacies, leave post-visit reviews, fill out structured questionnaires, and view ranked provider listings. The ranking is calculated automatically from questionnaire scores and star ratings.

**User Roles:**
- `patient` — Can search, review, and fill questionnaires
- `provider` — Doctor or pharmacy that can view their own profile/ratings
- `admin` — Can moderate reviews and manage the platform

## 3. Firebase Setup

### Step 1 — Create Firebase Project
1. Go to [https://console.firebase.google.com](https://console.firebase.google.com)
2. Create a new project named `healthcare-rating-app`
3. Disable Google Analytics (not needed)

### Step 2 — Enable Services
In the Firebase Console, enable the following:
- **Authentication** → Sign-in method → Enable `Email/Password` and `Google`
- **Firestore Database** → Create database → Start in **production mode**
- **Storage** → Start in production mode (for profile photos)

### Step 3 — Connect Flutter to Firebase
Run the following commands in the project root:
```bash
dart pub global activate flutterfire_cli
flutterfire configure
```
This auto-generates `lib/firebase_options.dart`.

```dart

  WidgetsFlutterBinding.ensureInitialized();

    options: DefaultFirebaseOptions.currentPlatform,
```

## 4. Database Schema (Firestore)

Firestore is a NoSQL document database. Data is organized into **collections** containing **documents**.

### Collection: `users`
```
users/{userId}
├── uid              : String   — Same as Firebase Auth UID
├── fullName         : String
├── email            : String
├── role             : String   — "patient" | "provider" | "admin"
├── profilePhotoUrl  : String   — Firebase Storage URL (nullable)
├── bookmarks        : Array    — List of provider document IDs
└── createdAt        : Timestamp
```

### Collection: `providers`
```
providers/{providerId}
├── providerId       : String   — Auto-generated Firestore ID
├── type             : String   — "doctor" | "pharmacy"
├── name             : String
├── specialty        : String   — e.g. "Cardiology", "General Practice"
├── address          : String
├── phone            : String
├── location         : GeoPoint — Latitude/Longitude
├── photoUrl         : String
├── averageRating    : Number   — Stored denormalized for fast reads (0.0–5.0)
├── totalReviews     : Number
├── rankingScore     : Number   — Composite score used for ranked listing
└── ownerId          : String   — Firebase Auth UID of the provider account (nullable)
```

### Collection: `reviews`
```
reviews/{reviewId}
├── reviewId         : String   — Auto-generated Firestore ID
├── providerId       : String   — Reference to providers/{providerId}
├── userId           : String   — Reference to users/{userId}
├── userName         : String   — Denormalized for display (avoids extra read)
├── overallRating    : Number   — 1–5 stars
├── comment          : String
├── questionnaire    : Map
│   ├── waitingTime      : Number  — 1–5
│   ├── serviceQuality   : Number  — 1–5
│   ├── hygiene          : Number  — 1–5
│   └── staffCommunication: Number — 1–5
├── isVerified       : Boolean  — True if visit was verified
└── createdAt        : Timestamp
```

### Collection: `questionnaires` (optional standalone, if needed separately)
```
questionnaires/{questionnaireId}
├── providerId       : String
├── userId           : String
├── waitingTime      : Number   — 1–5
├── serviceQuality   : Number   — 1–5
├── hygiene          : Number   — 1–5
├── staffCommunication: Number  — 1–5
└── submittedAt      : Timestamp
```

## 5. Firestore Security Rules

These rules are written in the Firebase Console under **Firestore → Rules**. They are the first line of security — never skip this step.

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // USERS — Only the user themselves can read/write their own document
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }

    // PROVIDERS — Anyone logged in can read. Only admins or the owner can write.
    match /providers/{providerId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null &&
        (get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin' ||
         resource.data.ownerId == request.auth.uid);
    }

    // REVIEWS — Anyone logged in can read. Only patients can create.
    // Only the review author or admin can delete.
    match /reviews/{reviewId} {
      allow read: if request.auth != null;
      allow create: if request.auth != null &&
        get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'patient';
      allow delete: if request.auth != null &&
        (resource.data.userId == request.auth.uid ||
         get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin');
      allow update: if false; // Reviews cannot be edited after submission
    }

    // QUESTIONNAIRES — Same rules as reviews
    match /questionnaires/{questionnaireId} {
      allow read: if request.auth != null;
      allow create: if request.auth != null &&
        get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'patient';
      allow update, delete: if false;
    }
  }
}
```

## 7. Services Layer

Services are pure Dart classes with no UI. They only talk to Firebase.

### `lib/services/auth_service.dart`
```dart
import 'package:firebase_auth/firebase_auth.dart';
import 'package:cloud_firestore/cloud_firestore.dart';
import '../models/user_model.dart';

class AuthService {
  final FirebaseAuth _auth = FirebaseAuth.instance;
  final FirebaseFirestore _db = FirebaseFirestore.instance;

  // Current user stream — listened to by AuthProvider
  Stream<User?> get authStateChanges => _auth.authStateChanges();

  // Sign Up
  Future<UserModel?> signUp({
    required String email,
    required String password,
    required String fullName,
    required String role,
  }) async {
    try {
      final credential = await _auth.createUserWithEmailAndPassword(
        email: email,
        password: password,
      );
      final user = UserModel(
        uid: credential.user!.uid,
        fullName: fullName,
        email: email,
        role: role,
      );
      await _db.collection('users').doc(user.uid).set(user.toMap());
      return user;
    } catch (e) {
      rethrow;
    }
  }

  // Sign In
  Future<UserCredential> signIn({
    required String email,
    required String password,
  }) async {
    return await _auth.signInWithEmailAndPassword(
      email: email,
      password: password,
    );
  }

  // Sign Out
  Future<void> signOut() async {
    await _auth.signOut();
  }

  // Fetch user data from Firestore
  Future<UserModel?> getUserData(String uid) async {
    final doc = await _db.collection('users').doc(uid).get();
    if (doc.exists) {
      return UserModel.fromMap(doc.data()!);
    }
    return null;
  }
}
```

### `lib/services/firestore_service.dart`
```dart
import 'package:cloud_firestore/cloud_firestore.dart';
import '../models/provider_model.dart';
import '../models/review_model.dart';

class FirestoreService {
  final FirebaseFirestore _db = FirebaseFirestore.instance;

  // ── PROVIDERS ──────────────────────────────────────────

  // Get top-ranked providers for home screen
  Future<List<ProviderModel>> getTopProviders({String? type}) async {
    Query query = _db
        .collection('providers')
        .orderBy('rankingScore', descending: true)
        .limit(10);
    if (type != null) {
      query = query.where('type', isEqualTo: type);
    }
    final snapshot = await query.get();
    return snapshot.docs
        .map((doc) => ProviderModel.fromMap(doc.id, doc.data() as Map<String, dynamic>))
        .toList();
  }

  // Search providers by name
  Future<List<ProviderModel>> searchProviders(String query) async {
    final snapshot = await _db
        .collection('providers')
        .where('name', isGreaterThanOrEqualTo: query)
        .where('name', isLessThanOrEqualTo: '$query\uf8ff')
        .get();
    return snapshot.docs
        .map((doc) => ProviderModel.fromMap(doc.id, doc.data()))
        .toList();
  }

  // Get single provider by ID
  Future<ProviderModel?> getProvider(String providerId) async {
    final doc = await _db.collection('providers').doc(providerId).get();
    if (doc.exists) {
      return ProviderModel.fromMap(doc.id, doc.data()!);
    }
    return null;
  }

  // ── REVIEWS ────────────────────────────────────────────

  // Get all reviews for a provider
  Future<List<ReviewModel>> getReviews(String providerId) async {
    final snapshot = await _db
        .collection('reviews')
        .where('providerId', isEqualTo: providerId)
        .orderBy('createdAt', descending: true)
        .get();
    return snapshot.docs
        .map((doc) => ReviewModel.fromMap(doc.id, doc.data()))
        .toList();
  }

  // Submit a new review
  Future<void> submitReview(ReviewModel review) async {
    final batch = _db.batch();

    // 1. Add the review document
    final reviewRef = _db.collection('reviews').doc();
    batch.set(reviewRef, review.toMap());

    // 2. Update provider's averageRating and totalReviews atomically
    final providerRef = _db.collection('providers').doc(review.providerId);
    batch.update(providerRef, {
      'totalReviews': FieldValue.increment(1),
      // NOTE: averageRating and rankingScore are recalculated in RankingService
    });

    await batch.commit();
  }

  // Bookmark a provider
  Future<void> toggleBookmark(String userId, String providerId, bool add) async {
    await _db.collection('users').doc(userId).update({
      'bookmarks': add
          ? FieldValue.arrayUnion([providerId])
          : FieldValue.arrayRemove([providerId]),
    });
  }
}
```

### `lib/services/ranking_service.dart`
```dart
import 'package:cloud_firestore/cloud_firestore.dart';
import '../models/review_model.dart';

class RankingService {
  final FirebaseFirestore _db = FirebaseFirestore.instance;

  /// Recalculates and updates the ranking score for a provider.
  /// Call this after every new review submission.
  ///
  /// Formula:
  /// rankingScore = (averageOverallRating * 0.4) +
  ///               (averageQuestionnaireScore * 0.6)
  Future<void> recalculateRanking(String providerId) async {
    final snapshot = await _db
        .collection('reviews')
        .where('providerId', isEqualTo: providerId)
        .get();

    if (snapshot.docs.isEmpty) return;

    final reviews = snapshot.docs
        .map((doc) => ReviewModel.fromMap(doc.id, doc.data()))
        .toList();

    final double avgOverall =
        reviews.map((r) => r.overallRating).reduce((a, b) => a + b) /
            reviews.length;

    final double avgQuestionnaire = reviews.map((r) {
          final q = r.questionnaire;
          final values = q.values.toList();
          return values.reduce((a, b) => a + b) / values.length;
        }).reduce((a, b) => a + b) /
        reviews.length;

    final double rankingScore =
        (avgOverall * 0.4) + (avgQuestionnaire * 0.6);

    await _db.collection('providers').doc(providerId).update({
      'averageRating': avgOverall,
      'rankingScore': rankingScore,
    });
  }
}
```

## 8. Providers (State Management)

Providers use the `provider` package. They listen to services and notify the UI of state changes.

### `lib/providers/auth_provider.dart`
```dart
import 'package:flutter/material.dart';
import 'package:firebase_auth/firebase_auth.dart';
import '../services/auth_service.dart';
import '../models/user_model.dart';

class AuthProvider extends ChangeNotifier {
  final AuthService _authService = AuthService();

  User? _firebaseUser;
  UserModel? _userModel;
  bool _isLoading = false;

  User? get firebaseUser => _firebaseUser;
  UserModel? get userModel => _userModel;
  bool get isLoading => _isLoading;
  bool get isLoggedIn => _firebaseUser != null;

  AuthProvider() {
    // Listen to auth state from Firebase
    _authService.authStateChanges.listen((user) async {
      _firebaseUser = user;
      if (user != null) {
        _userModel = await _authService.getUserData(user.uid);
      } else {
        _userModel = null;
      }
      notifyListeners();
    });
  }

  Future<void> signUp({
    required String email,
    required String password,
    required String fullName,
    required String role,
  }) async {
    _isLoading = true;
    notifyListeners();
    try {
      await _authService.signUp(
        email: email,
        password: password,
        fullName: fullName,
        role: role,
      );
    } finally {
      _isLoading = false;
      notifyListeners();
    }
  }

  Future<void> signIn(String email, String password) async {
    _isLoading = true;
    notifyListeners();
    try {
      await _authService.signIn(email: email, password: password);
    } finally {
      _isLoading = false;
      notifyListeners();
    }
  }

  Future<void> signOut() async {
    await _authService.signOut();
  }
}
```

```dart
  bool get isLoading => _isLoading;

  Future<void> loadHomeData() async {
    _isLoading = true;
    notifyListeners();
    _topDoctors = await _firestoreService.getTopProviders(type: 'doctor');
    _topPharmacies = await _firestoreService.getTopProviders(type: 'pharmacy');

    notifyListeners();

    if (query.isEmpty) {
      _searchResults = [];
      notifyListeners();
      return;
    }
    _isLoading = true;
    notifyListeners();
    _searchResults = await _firestoreService.searchProviders(query);
    _isLoading = false;
    notifyListeners();
  }
```

```dart
  @override
  void initState() {
    super.initState();
    _redirect();
  }

  Future<void> _redirect() async {
    await Future.delayed(const Duration(seconds: 2));
    if (!mounted) return;
    final auth = context.read<AuthProvider>();
    if (auth.isLoggedIn) {
      Navigator.pushReplacementNamed(context, AppRoutes.home);
    } else {
      Navigator.pushReplacementNamed(context, AppRoutes.onboarding);
    }
  }

  @override
  Widget build(BuildContext context) {
    return const Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Icon(Icons.local_hospital, size: 80, color: Colors.teal),
            SizedBox(height: 16),
            Text('HealthRate', style: TextStyle(fontSize: 28, fontWeight: FontWeight.bold)),
          ],
        ),
      ),
    );
  }
}
```
